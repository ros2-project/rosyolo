# ROS2 기반 YOLO 객체 인식 주행 로봇 (팀 프로젝트)
본 저장소에는 2024년 10월까지 진행된 프로젝트 코드가 포함되어 있습니다.
이후 12월까지 추가 개발을 진행하였으나, 로컬 환경 문제로 인해
해당 기간의 일부 코드가 보존되지 못했습니다.

현재 저장소의 코드는 프로젝트 진행 과정 및 구현 구조를 증빙하기 위한 용도로
업로드되었습니다.

## 개요
본 프로젝트는 라즈베리파이에서 YOLO 기반 객체 인식을 수행하고,
ROS2를 통해 STM32 제어 노드와 통신하여 로봇 주행을 제어하는 시스템입니다.

## 보존되지 못한 기능
1. 공과의 거리에 따른 속도 제어
2. 탁구공과 테니스공 구분 및 이에 따른 속도 제어
3. 공 수거 후 추가적인 공 탐색을 위한 장치의 제자리 회전 기능
4. 공이 장치의 수거범위 안으로 들어오면 서보모터를 통한 공 수거 기능
5. 각 노드들의 파라미터 값을 실험을 통한 수정 결과
